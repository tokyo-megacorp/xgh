# xgh Implement Design Workflow

Translate approved design artifacts into implementation plans and code tasks.

## Flow

1. Parse design goals, constraints, and acceptance criteria.
2. Break down design into incremental implementation tasks.
3. Map each task to files, interfaces, and tests.
4. Implement with verification checkpoints.
5. Curate design decisions and trade-offs.

## Deliverables

- design-to-task mapping
- implementation checklist
- validation notes
