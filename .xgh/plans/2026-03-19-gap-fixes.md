# xgh Gap Fixes Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix 5 gaps identified in the 2026-03-19 session: scheduler never starts, dual memory hooks, stale cursors, retention not enforced, custom skills have no trigger.

**Architecture:** Remove env var gates from session-start hook, add deterministic bash for retention/cursors, extend cron registration with configurable jobs. All changes are in hooks, skills (markdown), and one new bash script.

**Tech Stack:** Bash hooks, Markdown skills, YAML config. Test suite uses `assert_*` helpers.

**Spec:** `.xgh/specs/2026-03-19-xgh-gap-fixes-design.md`

Create `plugin/scripts/update-cursor.sh`:

```bash
#!/usr/bin/env bash
set -euo pipefail

# Usage: update-cursor.sh <channel-id> <timestamp>
# Atomically updates the cursor for a channel in ~/.xgh/inbox/.cursors.json

CURSORS_FILE="${HOME}/.xgh/inbox/.cursors.json"
CHANNEL="${1:?Usage: update-cursor.sh <channel-id> <timestamp>}"
TIMESTAMP="${2:?Usage: update-cursor.sh <channel-id> <timestamp>}"

# Ensure directory and file exist
mkdir -p "$(dirname "$CURSORS_FILE")"
[ -f "$CURSORS_FILE" ] || echo '{}' > "$CURSORS_FILE"

# Atomic update: write to temp, then move
TMP="$(mktemp)"
jq --arg ch "$CHANNEL" --arg ts "$TIMESTAMP" '.[$ch] = $ts' "$CURSORS_FILE" > "$TMP"
mv "$TMP" "$CURSORS_FILE"
```

Make it executable:
```bash
chmod +x plugin/scripts/update-cursor.sh
```
