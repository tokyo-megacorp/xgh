# /xgh-doctor — Pipeline Health Check

Run the `xgh:ingest-doctor` skill to validate the full ingest pipeline.

## Usage

```
/xgh-doctor
```

Outputs a structured ✓/✗ report with specific fix suggestions for any failures.
