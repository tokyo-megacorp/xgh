# xgh Investigate Workflow

Use this skill for structured incident and requirement investigation with MCP-aware discovery.

## MCP-Aware Investigation

1. Identify available MCP servers and relevant domains.
2. Gather logs, configs, and historical decisions from memory/context tree.
3. Reconstruct timeline and isolate likely root causes.
4. Produce hypothesis list with evidence quality.
5. Recommend next experiments or implementation path.

## Output

- Investigation summary
- Evidence table
- Ranked hypotheses
- Suggested follow-up actions
